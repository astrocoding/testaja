Changelog
Ringkasan revisi terbaru terhadap file yang diupload.

Revisi ini — berdasarkan Master_Pricing_untuk_AI.xlsx

Sumber data (bukan lagi kosong / bukan data contoh):

  • 145 Material real (66 Digital A3+ Full Color/BW + 79 Large Format) diimpor langsung dari Master_Pricing_untuk_AI.xlsx.
  • 1.673 baris Pricing Tier (range qty × profile × sisi/warna untuk A3+; per-lokasi untuk LF) diambil apa adanya dari sheet Print A3+ Full Color, Print A3+ Hitam Putih, Print LF Purwakarta, Print LF In House Karawang, dan Print LF Outsourcing.
  • 44 Lebar Bahan dan 4 Finishing (Laminating Cold LF) ikut terbawa dari sheet yang sama.
  • Ongkos Potong Digital A3+ (53 baris, per kelas ukuran A3/A4/A5/≤A6 + minimum) dipakai langsung untuk opsi cutting "Potong".

Perubahan logika:

  • Data Contoh/dummy dihapus total — tidak ada lagi tombol "Muat Data Contoh".
  • Large Format sekarang dihitung per meter linear roll (bukan per m²), sesuai kolom "Per Meter" di workbook sumber.
  • A3+ Laminasi (Glossy/Doff, 1/2 sisi) dan Cutting (Kiss Cut/Die Cut/Perforasi) memakai baris harga qty-range asli dari sheet, bukan angka flat.
  • Cutting "Potong" memakai tabel Ongkos Potong (kelas ukuran + minimum charge per kategori), hanya aktif di mode Hitung Layout.
  • Kiss Cut/Die Cut dibatasi hanya untuk material Sticker, sesuai data sumber.
  • Lokasi LF "Print LF In House Karawang" dipetakan ke entitas Graha atas instruksi langsung product owner.

Fitur baru:

  • Column Mapping pada Import Data — kolom file Anda dicocokkan ke field target (bukan dipaksa mengikuti skema baru).
  • Pricing import mendukung format lebar (Harga Umum / Harga Divisi / Harga Polos sebagai kolom terpisah pada baris yang sama).
  • Logo Cipta Grafika di kiri atas sidebar.
  • Sidebar tetap berbasis teks di semua ukuran layar; pada layar sangat kecil (≤560px) menjadi drawer yang bisa dibuka/tutup, teks tetap utuh.

Tidak berubah: struktur Internal Estimate / Customer Summary, Production Job Ticket tetap tidak dibuat (tanggung jawab ERP), fokus tetap hanya Digital Print A3+ dan Large Format.
